# 情報可視化
練習の段階から始め、最終的にはオープンデータを可視化する。

オープンデータとして、
- 新型コロナウイルス感染者数
- 地球温暖化データ
- 梅雨の入明データ
を用いた。

また、最終週では、地球温暖温暖化と梅雨の入明の関係性を示した。

## 実装
可視化のための主な方法として、[d3.js](https://d3js.org/)を利用した。
